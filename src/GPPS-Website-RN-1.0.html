Release Notes - GPPS Website v1.0

By Austin Musoke

We are excited to announce the release of the GPPS Website version 1.0. This release includes the following features:

This website boasts a variety of features designed to enhance user experience and accessibility, including a responsive sidebar that adapts to different screen sizes, ensuring easy navigation on both desktop and mobile devices. It also includes a smooth horizontal scrolling mechanism that allows users to browse through content effortlessly. Additionally, multiple links, including those in the footer (such as accreditation and registration), provide access to embedded PDF files for detailed information.

  • Home Button: A convenient home button that allows users to navigate back to the news and events page.
  • Image Display: A section dedicated to displaying images with descriptive captions.
  • Article Header: A header section that includes the title of the article and the author's name.
  • Responsive Design: The website is designed to be responsive and accessible on various devices.
  • Typography: Integration with Google Fonts and Fontshare for enhanced typography.
  • Styling: Custom CSS styles for a consistent and visually appealing layout.

Looking ahead, we are planning to introduce a "Contact Us" page that will allow users to reach out to us directly through the website. This page will be connected to a database to store and manage user inquiries efficiently. We are currently evaluating several database providers to ensure the best performance and reliability. The companies under consideration include MongoDB, PostgreSQL, and MySQL. Each of these options offers unique advantages, and we are committed to selecting the one that best meets our needs.

Additionally, we are looking to improve the horizontal sidebar on the homepage to provide a more seamless and intuitive navigation experience. This enhancement will make it easier for users to access different sections of the website quickly and efficiently.

We hope you enjoy the new features and improvements in this release. Thank you for your continued support.
